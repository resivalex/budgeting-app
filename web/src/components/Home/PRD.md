# Feature: Home

## Overview

Dashboard providing a visual overview of all user accounts with their current balances, using color-coded account cards for quick identification.

## Functionality

- **Account Balance Overview**: Displays all accounts with their current balance in a card layout
- **Color-Coded Account Cards**: Each account shows its associated color for visual identification
- **Multi-Currency Display**: Proper formatting and currency symbol display for each account currency
- **Responsive Card Layout**: Mobile-friendly two-column layout showing account name and balance

The home screen is the primary landing page providing users with immediate visibility into their financial position across all accounts.
